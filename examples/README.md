## Contributing an example

Each example lives in its own subdirectory with:
- `README.md` — clone, install, run instructions
- `.env.example` (no real secrets!)
- Working code

See [CONTRIBUTING.md](../CONTRIBUTING.md) at the repo root.

In the meantime, **[forum-coverage](https://pay.aizim.co.zw/forum-coverage)** documents how to wire ManishaPay into specific stacks even without a dedicated example.
